# Proyecto de Katas en Python

Este proyecto consiste en la resolución de múltiples Katas en Python diseñado para practicar y reforzar conceptos fundamentales de programación y análisis de datos.

## Objetivos del proyecto

A lo largo del proyecto se busca que el desarrollador demuestre conocimiento y manejo de los siguientes conceptos:

* **Tipos de datos básicos y funciones incorporadas:** cadenas, números, listas, tuplas, diccionarios y sus operaciones.
* **Estructuras de datos en Python y sus métodos:** uso de `list`, `set`, `dict`, `tuple`, y funciones como `map`, `filter`, `reduce`.
* **Condicionales:** `if`, `elif`, `else` para controlar el flujo del programa según las condiciones.
* **Estructuras de iteración:** bucles `for` y `while` para recorrer datos y realizar cálculos.
* **Funciones en Python:** definición de funciones, paso de argumentos, valores de retorno y funciones lambda.
* **Clases y Programación Orientada a Objetos (POO):** creación de clases, métodos, herencia, `super()`, atributos y encapsulación.
* **Uso de módulos:** importar y utilizar módulos como `functools` y `collections`.
* **Buenas prácticas de programación:** código legible, modular, uso de comentarios explicativos y manejo de excepciones.
